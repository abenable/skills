# PerfTest

Performance testing and benchmarking tool. Measure command execution time, benchmark disk IO, test network throughput, compare command performance, and generate reports.

## Quick Start

Run `perftest help` for available commands and usage examples.

## Features

- Fast and lightweight — pure bash with embedded Python
- No external dependencies required
- Data stored locally in `~/.perftest/`
- Works on Linux and macOS

## Usage

```bash
perftest help
```
